Keeper App

The Keeper App is a useful, everyday tool that can be used to keep notes on the web. On the app, to-do tasks can be written out and broken down into descriptions. Simple class notes could be jotted down as well with this title and description format. It can be treated similarly to a digital version of sticky notes. The app is built with React.js.

Fineapple Studios

This website was built to establish an idea for a music label called "Fineapple Studios". The idea started with 3 college classmates, including myself, who had grown up with a love for music, particularly music with Asian-American cultural influence. The vision of the label is to form a collective of young, upcoming artists that seek to build on their creative capacity and curate relevant experiences that contribute to the culture. The website was made with the use of Bootstrap and Node.js, and it currently only serves as a template for further expansion for the label's content.

Simon Game

A test of memory and attention, the Simon game will take you on a fun ride to see if you can follow along. Select the any of the four squares in the order that appear, from green, yellow, red, and to blue, for as many levels as you can. This projected was created as I learned to implement JavaScript logic and data structures with HTML/CSS.
